# Merge sort

My C++ implementation of merge-sort, fully commented.

## Run with Clion

 1. Open with Clion
 2. Build the project
 3. Run the project

## Run with g++

 1. `cd` to the project directory
 2. `g++ *.h && g++ *.cpp` to compile the project
 3. `./a.out` to run the project
